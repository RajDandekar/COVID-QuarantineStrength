Global quarantine strength and reproduction number comparison

Comparison of the quarantine strengrh function and the effective reproduction variation in several countries is estimated. The method followed is based on a Physics Informed Neural Network (PINN): an augmentation of the standard SIR epidemiological model with machine learning. The physics informed neural network model is universally applicable making it a flexible and powerful tool to analyze quantities like quarantine strength and the effective reproduction number temporal evolution.
